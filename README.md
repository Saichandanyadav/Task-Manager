## Task Manager App

This is a simple task management app built using **React** and **Tailwind CSS**.

I created this project to understand how React works and how to store data using localStorage.

---

## What this app does

* Admin can create tasks
* Admin can assign tasks to users by drag and drop
* User can see only their tasks
* User can mark task as completed
* Data is saved in localStorage
* Data does not disappear on refresh

---

## Roles

### Admin

* Can add new tasks
* Tasks are unassigned at first
* Can drag tasks to users

### User

* Can see assigned tasks
* Can mark the tasks as completed

---

## Storage

* Tasks are stored in localStorage
* Users are stored in localStorage
* Logged in user is stored in localStorage
* Clearing localStorage will reset everything

---

## Technologies Used

* React
* Tailwind CSS
* JavaScript

---

## How to run

1. Open the project
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:5173/`

---

## Login

* Enter any name
* Choose Admin or User
* Only User login adds name to user list
* Admin login does not add to user list
