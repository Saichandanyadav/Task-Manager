## Task Manager App

This is a simple task management app built using **React** and **Tailwind CSS**.

I created this project to understand how React works and how to store data using localStorage.

**Live Demo:**
[https://chandan-task-manager.vercel.app/](https://chandan-task-manager.vercel.app/)

---

## What this app does

* Admin can create tasks
* Admin can assign tasks to users using drag and drop
* User can see only their assigned tasks
* User can mark tasks as completed
* Data is saved in localStorage
* Data does not disappear on page refresh

---

## Roles

### Admin

* Can add new tasks
* Tasks are unassigned initially
* Can drag and drop tasks to users

### User

* Can see assigned tasks
* Can mark tasks as completed

---

## Storage

* Tasks are stored in localStorage
* Users are stored in localStorage
* Logged in user is stored in localStorage
* Clearing localStorage will reset the app

---

## How to run

1. Open the project folder
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:5173/` in browser

---

## Login

* Enter any name
* Select Admin or User
* Only User login adds the name to user list
* Admin login does not add to user list

---

## Purpose

This project is built to practice React basics like:

* useState
* props
* conditional rendering
* localStorage usage
